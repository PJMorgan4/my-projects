Scratch Game

We made a team project with the theme of Marvel. We had two weeks to create an entirely new quest game with more rooms and add-ons. The game takes Deadpool on a journey through time as he tries to return to the present. You can go to various points in time and beat different challenges. The hardest part was transferring pictures, code, and sprites from one teammate's computer to another. Another challenge was the time. I definitely felt the pressure at the end to finish with a tight deadline. In the end I think it was a good experience because it taught me how the basics of code work.
